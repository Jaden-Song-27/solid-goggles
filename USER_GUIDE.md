## 1. 安装与启动

### 环境要求

- Windows 10/11 (64位)
- Node.js 18+

### 启动方式

```bash
# 进入项目目录
cd smart-ime

# 安装依赖（仅首次）
npm install

# 启动开发模式
npm run dev
```

### 打包安装程序

```bash
npm run electron:build:win
```

打包完成后，在 `release/` 目录下找到 `SmartIME Setup.exe`，双击安装。

## 2. 基本操作

### 调出输入面板

| 方式 | 操作 |
|------|------|
| 托盘右键 | 右键右下角 SmartIME 图标 → "显示/隐藏输入面板" |
| 托盘单击 | 单击右下角 SmartIME 图标，直接切换显示/隐藏 |
| 快捷键 | 双击 Ctrl 键（部分系统可能不支持） |

### 面板说明

弹出后面板显示：

```
ni hao │ 1你好  2尼好  3你号  4拟好  5逆豪
       ↑ 闪烁光标
```

- **左侧**：当前输入的拼音（带闪烁光标）
- **分隔线**：竖线分隔拼音区和候选词区
- **右侧**：候选词列表，数字 `1-9` 对应各候选词

### 隐藏面板

- 点击面板外部区域（面板自动失焦隐藏）
- 或点击托盘图标
- 或按 `Esc` 键

## 3. 拼音输入

### 完整拼音

输入完整的拼音音节：

```
nihao     → 你好、尼好、你号...
shanghai  → 上海、伤害...
zhongguo  → 中国...
```

### 首字母缩写（简拼）

只输入每个字的首字母：

```
nh   → 你好、那会、年后...
sh   → 上海、社会、时候...
wm   → 我们、完美、文明...
```

### 选词操作

| 按键 | 功能 |
|------|------|
| `1-9` | 直接选择对应编号的候选词 |
| `空格` | 选择第一个（高亮）候选词 |
| `Enter` | 选择第一个（高亮）候选词 |
| `← → ↑ ↓` | 切换高亮的候选词 |
| `Tab` | 循环切换候选词 |
| `Backspace` | 删除最后一个拼音字母 |
| `Esc` | 取消输入，隐藏面板 |

### 鼠标操作

直接**点击**候选词即可选择上屏。

### 选词上屏

选择候选词后：
1. 面板播放淡出动画
2. 面板自动隐藏
3. 选中文字自动输入到当前光标位置

## 4. 快捷指令

在输入面板中直接输入以 `/` 开头的指令：

### 内置指令

| 指令 | 功能 | 示例 |
|------|------|------|
| `/date` | 插入当前日期 | `2026-06-02` |
| `/time` | 插入当前时间 | `15:30:45` 或 `15:30` |
| `/datetime` | 插入日期+时间 | `2026-06-02 15:30` |
| `/calc 表达式` | 计算数学表达式 | `/calc 23*47+15` → `1096` |
| `/trans 单词` | 英译中 | `/trans hello` → `你好` |
| `/clip` | 插入剪贴板内容 | 将剪贴板中的文字上屏 |

### 数学运算

无需 `/calc` 指令，直接输入数学表达式即可：

```
23*47+15   → 1096
(100-20)/4 → 20
2**10      → 1024 (指数)
```

支持的运算符：`+` `-` `*` `/` `%` `**` `()` 

### 自定义指令

参见 [快捷指令管理](#8-快捷指令管理)。

## 5. 后悔模式

后悔模式允许你**撤销刚才上屏的文字**，重新选择候选词。

### 触发方式

在上屏文字后的 **2 秒内**，再次调出输入面板（双击 Ctrl 或单击托盘），面板会自动进入后悔模式。

### 操作

| 操作 | 效果 |
|------|------|
| 自动 | 刚才上屏的文字被删除，恢复候选词列表 |
| 按 `1-9` | 选择新的候选词替换原来的 |
| 按 `Esc` | 取消后悔，恢复原来的文字 |

### 连续后悔

在后悔模式下再次按 `Ctrl+Z`，可以继续回退到更早的上屏文字（最多 50 条历史记录）。

### 限制

- 仅在上屏后 **2 秒内**有效
- 快捷指令（以 `/` 开头）不支持后悔
- 只有 1 个候选词的情况不支持后悔

## 6. 皮肤系统

SmartIME 内置 **13 套皮肤**，可通过托盘菜单实时切换。

### 内置皮肤列表

#### 经典皮肤（5套）

| 皮肤 | 风格 | 特点 |
|------|------|------|
| **Ghost 幽灵** | 暗色毛玻璃 | 半透明灰黑背景，默认皮肤 |
| **Terminal 终端** | 终端风格 | 纯黑背景 + 绿色等宽字体 |
| **Frosted 冰霜** | iOS 风格 | 强毛玻璃 + 大圆角，浅色 |
| **Ink 水墨** | 中式风格 | 米白背景 + 衬线字体 + 竖排候选 |
| **Minimal 极简** | 极简风格 | 完全无背景，纯文字悬浮 |

#### 彩色玻璃皮肤（8套）

| 皮肤 | 颜色 | 特点 |
|------|------|------|
| **紫水晶** | 紫色 | 30px 毛玻璃 + 饱和度增强 |
| **深海蓝** | 蓝色 | 半透明深海蓝背景 |
| **翡翠绿** | 绿色 | 半透明翡翠绿背景 |
| **玫瑰粉** | 粉色 | 半透明玫瑰粉背景 |
| **日落橙** | 橙色 | 半透明日落橙背景 |
| **午夜黑** | 深色 | 深色半透明背景 |
| **天空蓝** | 浅蓝 | 半透明天空蓝背景 |
| **薄荷绿** | 青绿 | 半透明薄荷绿背景 |

所有彩色玻璃皮肤均具有：
- 30px 强毛玻璃模糊效果
- 180% 色彩饱和度增强
- 半透明边框

### 切换皮肤

**右键托盘图标 → 皮肤 → 选择想要的皮肤**，即时切换生效。

### 导入/导出皮肤

通过皮肤编辑器（见[下一节](#7-皮肤编辑器)）可以：
- **导出**：将当前皮肤保存为 `.smartskin.json` 文件
- **导入**：加载他人分享的皮肤文件
- **拖放导入**：直接将 `.json` 或 `.skin.json` 文件拖到面板上

## 7. 皮肤编辑器

可视化 WYSIWYG 皮肤编辑器，所见即所得。

### 打开方式

**右键托盘图标 → 皮肤 → 皮肤编辑器**

或拖放皮肤文件到输入面板上。

### 编辑面板

编辑器左侧为**实时预览**区，右侧为**参数调整**区：

#### 基础设置

| 参数 | 说明 | 范围 |
|------|------|------|
| 圆角 | 面板圆角大小 | 0-30px |
| 内边距 | 面板内边距 | 0-40px |
| 透明度 | 面板整体透明度 | 20%-100% |
| 毛玻璃模糊度 | backdrop-filter 模糊强度 | 0-50px |
| 背景色 | 面板背景颜色 | 颜色选择器 |
| 阴影 | CSS box-shadow | 自由输入 |

#### 候选词

| 参数 | 说明 |
|------|------|
| 字号 | 候选词字体大小 (10-28px) |
| 间距 | 候选词之间的间距 (0-30px) |
| 圆角 | 候选词圆角 (0-20px) |
| 排列方向 | 横向 / 纵向 / 网格 |
| 普通/悬停/选中色 | 三种状态的背景和文字颜色 |

#### 动画

| 参数 | 选项 |
|------|------|
| 唤出动效 | 缩放 / 淡入 / 下滑 / 上滑 / 无 |
| 消失动效 | 淡出 / 缩放 / 上滑 / 下滑 / 无 |
| 动画时长 | 0-600ms |
| 皮肤切换过渡 | 交叉淡入淡出 / 即时切换 |

#### 高级 — 自定义 CSS

直接输入 CSS 规则，追加到皮肤样式末尾，用于微调细节。

### 操作按钮

| 按钮 | 功能 |
|------|------|
| **保存** | 保存对当前皮肤的修改 |
| **另存为** | 基于当前皮肤创建新皮肤 |
| **导入皮肤文件** | 从文件导入皮肤 JSON |
| **导出当前皮肤** | 导出当前皮肤为 JSON 文件 |
| **重置为默认** | 恢复皮肤到原始状态 |

## 8. 快捷指令管理

### 打开方式

**右键托盘图标 → 快捷指令管理**

### 内置指令

| 指令 | 说明 | 可禁用 |
|------|------|--------|
| `/date` | 当前日期 | ✅ |
| `/time` | 当前时间 | ✅ |
| `/datetime` | 日期+时间 | ✅ |
| `/calc` | 计算器模式 | ✅ |
| `/trans` | 英译中 | ✅ |
| `/clip` | 剪贴板内容 | ✅ |

每个内置指令可以通过勾选框**启用/禁用**。

### 自定义指令

#### 新建指令

1. 点击 **"+ 新建指令"** 按钮
2. 填写：
   - **触发词**：如 `/email`（自动补 `/` 前缀）
   - **描述**：简短描述指令用途
   - **模板内容**：输入触发词时上屏的文字
3. 点击 **保存**

#### 模板占位符

模板中可以使用以下占位符：

| 占位符 | 说明 | 示例 |
|--------|------|------|
| `$input` | 触发词后的所有输入 | `/email zhangsan` → `$input` = `zhangsan` |
| `$arg` | 同 `$input` | — |
| `$1` | 同 `$input` | — |

#### 示例

| 触发词 | 模板 | 输入 | 输出 |
|--------|------|------|------|
| `/email` | `$input@gmail.com` | `/email zhangsan` | `zhangsan@gmail.com` |
| `/sig` | `此致\n敬礼\n张三` | `/sig` | 多行签名 |
| `/addr` | `北京市朝阳区xxx号` | `/addr` | 固定地址 |

#### 编辑/删除

每个自定义指令支持**编辑**和**删除**操作。

## 9. 设置

### 打开方式

**右键托盘图标 → 设置**

### 设置项

| 设置 | 说明 | 默认值 |
|------|------|--------|
| **默认皮肤** | 应用启动时使用的皮肤 | Ghost |
| **自动隐藏延迟** | 面板无操作后自动隐藏的时间 | 3秒 |
| **显示皮肤标签** | 面板右上角显示当前皮肤名称 | 开启 |
| **开机自启** | 系统启动时自动运行 SmartIME | 关闭 |

### 自动隐藏

- 可设置 0-10 秒（设为 0 禁用自动隐藏）
- 隐藏前 0.8 秒面板会**呼吸闪烁**，提示即将隐藏
- 任何按键操作会**重置计时器**

## 10. 常见问题

### Q: 面板弹不出来？

1. 确认 SmartIME 在运行（右下角有托盘图标）
2. 右键托盘图标 → "显示/隐藏输入面板"
3. 若托盘也看不到，在终端运行 `npm run dev` 查看错误信息

### Q: 输入拼音后没有候选词？

- 确认输入的是有效拼音（不是英文单词）
- 尝试输入完整拼音而非简拼
- 检查是否误触了 `/` 进入快捷指令模式

### Q: 文字上屏后变成了乱码？

- 确认目标应用支持中文输入
- 部分老旧应用（如某些命令行工具）可能不支持 Unicode 注入

### Q: 双击 Ctrl 没反应？

- 部分系统/Electron 版本不支持 `ControlLeft`/`ControlRight` 分离
- 改用托盘图标操作（右键 → 显示/隐藏输入面板）

### Q: 如何卸载？

- 通过 Windows "设置 → 应用 → SmartIME → 卸载"
- 或运行安装目录下的 `Uninstall SmartIME.exe`

### Q: 如何备份自定义皮肤和指令？

自定义数据保存在：
- `%APPDATA%/SmartIME/skins/` — 自定义皮肤
- `%APPDATA%/SmartIME/commands.json` — 自定义指令
- `%APPDATA%/SmartIME/settings.json` — 设置

备份以上目录/文件即可。
