# SmartIME — 产品概述

## 产品定位

SmartIME 是一款**面向效率型用户的桌面端智能拼音输入法**，以"后悔模式"和"可编程皮肤"为核心差异化，切入被搜狗、讯飞等巨头把持的中文输入法市场。

**一句话：用 AI 的思维方式重新设计输入法——不只是打字，是每次输入都给你反悔和定制的权利。**

## 核心价值主张

| 传统输入法 | SmartIME |
|-----------|----------|
| 打错字 → 退格重打 | 打错字 → Ctrl+Z 一秒回退重选 |
| 换肤像换壁纸，只有观感变化 | 皮肤 = 可编程的交互方案，CSS 级定制 |
| 被动接受推荐 | 主动训练（使用频率自学习） |
| 功能封闭 | `/` 命令生态，开放扩展 |

## 目标用户

- **知识工作者**：程序员、产品经理、写作者，对输入效率和工具质感有要求
- **个性化需求用户**：喜欢定制工具外观和交互方式的人
- **AI 产品早期尝鲜者**：对 AI 赋能的生产力工具感兴趣的创新者

## 技术栈

| 层 | 技术 | 说明 |
|---|------|------|
| 桌面框架 | Electron 30 | 跨平台桌面应用 |
| 前端 | React 18 + TypeScript | 类型安全的 UI 开发 |
| 构建 | Vite 5 | 极速 HMR 开发体验 |
| 状态管理 | Zustand 4 | 轻量响应式状态 |
| 样式 | CSS Variables + 皮肤引擎 | 运行时换肤 |
| 文本注入 | koffi (Win32 FFI) | 模拟键盘输入到任意应用 |
| 打包 | electron-builder (NSIS) | Windows 安装程序 |

## 当前版本

- **版本号**：v0.1.0
- **平台支持**：Windows 10/11（x64）
- **字库规模**：5200+ 汉字，覆盖 GB2312 一级字库
- **皮肤数量**：5 套内置 + 无限自定义
