## 正文

最近把自己用了三年的输入法卸载了。

理由很简单——每次打完字都弹广告窗口，关掉一个又来一个。我想安安静静打几个字，为什么要被当成流量变现的工具？

于是花了两个月，自己写了一个。

### 它长什么样

按下 `Alt + =`，屏幕中间（光标附近）弹出一个半透明的小面板：

```
ni hao │ 1你好  2尼好  3你号  4拟好  5逆豪        [↵] [✕]
```

选词、上屏、面板消失。整个过程不到两秒。

### 它还能做什么

我把自己日常高频的几个动作直接做进了输入面板里：

- **数学直算**：输入 `23*47+15`，直接出 `1096`。不用切计算器
- **英译中**：输入 `/trans hello` 回车，上屏 `你好`
- **快捷指令**：`/date` 插日期、`/time` 插时间、`/clip` 贴剪贴板
- **自定义指令**：可以建一个 `/email`，打 `zhangsan` 自动补成 `zhangsan@gmail.com`

### 设计原则

1. **无广告，无追踪** — 所有数据都在本地，不需要登录
2. **出现就出现，消失就消失** — 不用的时候就缩在右下角托盘，不占地方
3. **好看** — 内置 8 套毛玻璃彩色皮肤，还带了一个可视编辑器

### 技术细节（挑感兴趣的看）

- 基于 Electron + React + TypeScript，MIT 开源
- 拼音引擎：自研贪心音节切分 + 本地 2500 字词库 + 频率自学习
- 文字注入：Win32 SendInput API，兼容绝大多数 Windows 应用
- 安全性：数学解析器是手写递归下降，没用 eval()；PowerShell 注入做了 Base64 转义

### 已知问题

- v0.2.0 还是早期版本，词库只有 2500 字，不如商业输入法
- 目前只支持 Windows（macOS 和 Linux 只做了基础适配）
- 全拼还行，简拼识别率可以再提升
