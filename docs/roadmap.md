# SmartIME 产品路线图

## v0.3.0（当前版本）✅ 已完成

- [x] 拼音输入（全拼 + 简拼，2500 字词库）
- [x] 快捷指令系统（/date、/time、/calc、/trans、/clip）
- [x] 自定义指令模板（$input 占位符替换）
- [x] 数学直算（递归下降解析器，支持 +-*/%**）
- [x] 后悔模式（2 秒内 Ctrl+Z 撤回重选）
- [x] 13 套内置皮肤（5 经典 + 8 彩色毛玻璃）
- [x] 可视化皮肤编辑器（WYSIWYG + 导入导出）
- [x] 记忆功能（最近 20 条上屏文字）
- [x] 自动隐藏（可配置 0-10s，呼吸闪烁提示）
- [x] 系统托盘常驻 + 右键菜单
- [x] 注入管线重写（AttachThreadInput + SetForegroundWindow + SendInput）
- [x] 安全加固（CSP 头部、无 eval()、Base64 PowerShell 编码）
- [x] Windows 安装包打包（NSIS，76MB）

## v0.4.0 计划中

- [ ] 云端 LLM Agent（/ai 指令，调用 API 进行文本润色、翻译增强）
- [ ] RAG 知识库插件（用户配置本地文档，面板内检索问答）
- [ ] 剪贴板历史管理（Alt+V 呼出最近复制记录）
- [ ] 自定义 Agent 工作流（/workflow 串联多个指令）
- [ ] 数据统计面板（输入频率、常用词、指令使用排行）
- [ ] macOS 完整适配（CGEvent 注入 + 权限申请）

## v0.5.0 展望

- [ ] 词库扩展至 GB2312 全量（6763 汉字） + 云词库同步（可选）
- [ ] 上下文感知（读取光标前文本，智能补全与联想）
- [ ] 多语言输入支持（英文、日文基础支持）
- [ ] 插件市场（社区贡献指令和皮肤）
- [ ] Linux 完整适配（Wayland + X11）

## 产品北极星指标

**面板唤醒次数 × 上屏成功率 × 用户留存天数**

当前聚焦：将上屏成功率从 ~85% 提升至 99%。
