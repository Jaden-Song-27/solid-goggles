## SmartIME 的差异化优势

### 1. 后悔模式 (Regret Mode)
传统输入法打错字后：删除 → 重新输入 → 重新选字
SmartIME：**Ctrl+Z → 回到候选列表 → 一键重选**

这背后是产品设计思维：**输入不是一个单向流水线，而是一个随时可以回溯的对话**。用户不应该为选错字付出"全部重来"的代价。

### 2. 可编程皮肤
竞品的"皮肤"只是换背景图。SmartIME 的皮肤 = 一套完整的交互方案：
- 候选词排列方向（横排/竖排/网格）
- 动画曲线（淡入/缩放/滑入/无动画）
- CSS 变量级定制，可注入自定义 CSS
- 导入/导出，社区共享

### 3. 快捷指令生态
`/date` `/time` `/calc` `/trans` `/clip` … 输入法不仅仅是打字，更是一个轻量级的工作流入口。用户可创建自定义指令，接入自己的工具链。

### 4. 完全本地化
不依赖云端词库，所有数据存储在本地。在隐私意识增强的趋势下，这是一个重要的差异化点。

## 市场机会

1. **AI 输入法窗口期**：大模型正在重构输入法赛道（如 ChatGPT 辅助写作），SmartIME 的架构支持未来接入 LLM
2. **垂类场景**：程序员、学术写作、外语学习等需要特殊输入体验的场景
3. **工具型产品社区化**：以皮肤/指令市场为纽带，建立开发者/设计师社区

## 风险与挑战

| 风险 | 应对 |
|------|------|
| 字库/词库不如竞品 | v0.5 接入开放词库，v1.0 接入 LLM |
| 缺少语音输入 | 暂不进入语音赛道，聚焦文字输入体验 |
| 腾讯/搜狗的资源碾压 | 不正面竞争，做垂类精品 |
| Electron 内存/性能 | 持续优化，保持轻量 |
